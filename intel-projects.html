Intelligence Projects

The below projects are a culmination of coursework, self-guided skill mastery, and questions I want to analize further.

Penn State SRA G-8 Simulation: Intelligence Report

This project is a counterterrorism simulation in which students act as intelligence analysts representing G-8 countries in the Roma-Lyon Group, analyzing weekly intelligence, media, and biographical reports to identify and predict a major terrorist plot. Each team contributes country-specific analysis while sharing summarized insights with other G-8 members, reflecting real-world intelligence coordination where no single actor has the full picture. The project emphasizes collaborative analysis, critical thinking, and turning incomplete information into actionable assessments, and it concludes with analytical reports, a final country-team report, a joint G-8 presentation, and peer evaluation.

View the full report (PDF)

Tools and Methods Used:

  • Course intelligence simulation reports
  • Media and U.S. intelligence reports
  • Biographical Reports Request Tool
  • Word processing software
  • Presentation software

OSINT Python Tools Project

This project focused on developing practical Python skills by developing a few open-source intelligence (OSINT) tools. Inspired by OSINT Techniques by Michael Bazzell, the goal was to move beyond abstract programming exercises and instead build tools that reflect real intelligence workflows. Although assigned as a group project, the work was completed independently and resulted in five functional OSINT tools covering IP geolocation, domain WHOIS analysis, social media scraping, email validation, and automated open-source searching. Each tool was intentionally designed with a real-world intelligence or cybersecurity use case in mind, demonstrating how open-source data can be collected, assessed, and applied to investigative or analytical scenarios. The project emphasizes hands-on learning, technical curiosity, and alignment with long-term goals of becoming an all-source intelligence analyst.

View the write-up (PDF)

Tools and Methods Used:

  • Python programming language
  • Requests library for web queries
  • BeautifulSoup for HTML parsing
  • WHOIS library for domain analysis
  • Regular expressions for pattern matching
  • Open-source web services and public data sources
